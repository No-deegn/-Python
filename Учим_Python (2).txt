Переменные в Python

Переменные в python создаются легко, нужно лишь написать что то - это название переменной. Переменная - это контейнер в котором мы храним какие либо данные(данные это цифры/текст без контекста).
Пример: Box = 2 или Num = 4. Дынными ты можешь как либо манипулировать: изменять, удолять. Переменные нужны для хранения каких либо данных на время их исполнения. Название пременной не может начинатся на цифры или спец-символы. Пример: 342Box или $Num. 
--------------------------------------------------------------
Типы данных в Python
Пока разберем 4 типа данных: number = 5, fnumber = 5.7, name = "Alex" status = true/false. Так же воспользуемся "Комментариями", комментарии создаются с помощью # 

#Целочисленное число(целое число)--interger(int)
number = 5

#Вещественное(дробное число)--float
fnumber = 5.7

#Строка(текст)--string(str)
name = "Alex"

#Булевое значение--bool
status = True/false

Комментарии можно использовать для обозначение кусочка кода для того кто будет читать код, или для себя пометить что то.
--------------------------------------------------------------
Опреатор вывода(функция вывода)--> print(). 
Print() нужен для вывода данных в консоль/терминал. В функции print() обязательно открываются и закрываются круглые скобки,а внутри этих скобак указываются "аргументы" функции.

Пример:

Print("Привет мир!")
___________
Привет мир!
___________

Также можно выводить числа:

Print(777)
___________
777
___________

И переменные(то что в них хронится)

number = 532

print(number)
___________
532
___________

Экранирование 

Давайте рассмотрим ситуацию если нам нужно добавть текст где у нас слово стоит в ковычках: Он "плохой" человек
Если мы напишем так print("Он "плохой" человек") строка заканчивается здесь:print("Он -->"<-- плохой" человек") все что дальше python не распознает, что слово "плохой" это какая то команда которой нет в python. и иза этого выдаст ошибку invalid syntax. Тут уже нам нужно экранирование.

Пример экранирования:

print("Он \"плохой\"человек") #нужно поставить бэк-слэш преред (") -->(\").То есть СТРОКА заканчивается на ковычке после слова "человек-->"<-- , а не после слова ("он").
Альтернативный вариант, можно просто поместить слово -->полохой<-- в одинарные ковычки 'плохой'.

Пример:

print("он'плохой'человек") #и наоборот print('он"плохой"человек')
____________
Он "плохой" человек
____________
--------------------------------------------------------------
Перевод строки 

Мы пишем print("Привет мир") и нам нужно чтобы слово "мир" был на тругой строке. Нам нужен спец-символ перевод строки (\n), этот спец-символ(\n) нужно поместить перед словом которое нужно перенести на следующию строку.

Пример:

print("Привет\nмир")
____________
Привет
мир
____________

Конкатенация

Расмотрим пример: нам нужно вывести имя человека, но не самим написать его а взять его из пременной, и тут нам нужна конкатенация.

Пример:

user = "Alex"

print("Здраствуйте, " + user) #Мы поставили + между строкой и переменной - это и есть конкатенация, мы соеденяем две эти части. если нам нужно еще что то добавить (строку, цыфры, данные с переменной) ставим между ними + .
____________
Здравствуйте, Alex
____________

Есть правило: нельля конкатенировать разные типы данных. Переменная age = 18 хранит число (int), и мы не можем его соеденить(конкатенизировать) с другими данными, например: 

age = 18

print("Мне "+ age + "лет")
____________
TypeError: can only concatenate str (not int) to str (TypeError: можно объединять только str (не int) в str)
____________  

Что бы использовать данные с перемненой age = 18 - тип данных int, нам нужно перевести данные в строку(str) c помщью функции: str()

Пример:

age = 18

print("Мне "+ str(age) + "лет")
____________
Мне 18 лет
____________

Простые операции и функция input

Функция input
Если вы например хотите запросить у пользователя данные например его имя, то мы используем фукцию input().

#Создаем пременную используя функцию input()
name = input("Введите свое имя: ")
age = input("Укажите свой возраст: ")

Print("Здравствуйте, " + name + "!")
Print("Вам уже " + age + " года, это круто!!!")
____________
Введите свое имя: Саша
Укажите свой возраст: 23
Здравствуйте, Саша!
Вам уже 23 года, это круто!!!
____________

Здесь мы не стали использовать функцию scr(), чтобы перевести данные в переменой age потому, что что бы не ввел в запрос(Укажите свой возраст: или другие запросы сделанные с помощью функии input() ) пользоватьель это будет строкой(scr).
